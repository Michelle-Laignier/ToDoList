# ToDo List &#128203;
ToDo List app made with React Native. <br>
Features: <br>
- Create task <br>
- Check and uncheck a task as completed <br>
- Delete a task from the list <br>
- Show number of completed tasks

## Techs &#9883;
- React Native <br>
- TypeScript <br>
- Expo <br>
- [React Native Bouncy Checkbox](<https://github.com/WrathChaos/react-native-bouncy-checkbox>)
